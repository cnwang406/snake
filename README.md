# snake


![](https://img.shields.io/badge/python-3.9-blue)


simple snake, by python 3.9
import curses for text/Ascii mode. No color now.

to increase initial speed, modify
`SPEED_STEPS `
and 
`SPEED_FACTOR`

# to INSTALL:
```
pip3 install keyboard
pip3 install numpy
```
then run
```
python main.py
```
